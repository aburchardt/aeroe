På siden bliver der gjort brug af fonten Barlow Condensed. Der bliver gjort brug af fontens weight på 700, 600 og 400, samt 400i. Det er fordelt på de følgende tags på denne måde:
400 til "p" og "i" tags, 600 til "h4" tags, og 700 til "h3", "h2" og "h1" tags.
